Membuat Shopping Cart Dengan CodeIgniter Part 1 | CodeIgniter

KiosCoding CODEIGNITER 4/23/2018
-Cara Menggunakan Cart Di Codeigniter. Assalamualaikum wr.wb, pada tutorial kali ini kita akan belajar bagaimana cara membuat shopping cart sederhana menggunakan framework codeigniter. Apa sih cart itu? cart adalah sebuah keranjang belanja pada sebuah aplikasi, cart ini sering kita jumpai pada aplikasi-aplikasi E-Commerce atau online shop.

Ketika kita ingin membuat sebuah aplikasi berjenis e-commerce maka shopping cart ini wajib ada pada aplikasi kita. Nah pada tutorial kali ini kita akan belajar membuat nya menggunakan library cart codeigniter. Sebelum kita masuk pada tutorial saya ingin menjelaskan terlebih dahulu apa saja yang akan kita buat pada tutorial shopping cart kali ini.

Pada tutorial kali ini kita akan belajar :
  1. Menambahkan barang atau produk ke cart.
  2. Melihat daftar produk yang ada di cart.
  3. Mengubah atau menambah Qty pada cart.
  4. Menghapus produk yang ada di cart.
  5. Memasukan semua produk yang ada di dalam cart kedalam database.

Nah sekarang kita akan belajar bagaimana cara menambahkan barang atau produk kedalam cart dan melihat barang atau produk yang ada di dalam cart.

Mengaktifkan Library Yang Akan Digunakan 

Setelah kita membuat database dan tabel baru, sekarang kita akan mengaktifkan atau me-load librrary yang akan digunakan, karena kita akan berhubungan dengan database dan cart maka kita akan mengaktifkan library database nya. application/config/autoload.php.




$autoload['libraries'] = array('database','cart');

Dan jangan lupa untuk mengaktifkan helper url nya juga.



$autoload['helper'] = array('url');

Setelah mengaktifkan library serta helper yang dibutuhkan, sekarang kita setting base_url di config.php dengan url project sobat. sebagai contoh disini saya mengisikan nya sebagai berikut.



$config['base_url'] = 'http://localhost/kioscoding/';


 

Membuat Controller Baru

Langkah awal yang harus dilakukan setelah kita mensetting konfigurasi dasar adalah membuat controller baru dengan nama Shopping_cart.php. Disini kita akan membuat 3 function sekaligus yaitu function index() yang digunakan untuk menampilkan daftar barang yang ada di database, lalu function beli() yang akan digunakan untuk memasukan barang yang ingin dibeli ke dalam cart, dan function keranjang_belanja() yang akan digunakan untuk menampilkan daftar barang yang disimpan didalam cart.

Baiklah setelah kita mengetahui struktur kode yang akan dibuat sekarang buatlah sebuah controller baru dengan nama Shopping_cart.php kemudian salin lah baris kode berikut dibawah ini.



<?php
defined('BASEPATH') OR exit('No direct script access allowed');

class Shopping_cart extends CI_Controller{

  public function __construct()
  {
    parent::__construct();
    $this->load->model('M_cart');
  }

  public function index(){

    $data['product'] = $this->M_cart->get()->result();
    $data['cart'] = $this->cart->contents();
    $this->load->view('index',$data);
  }

  public function beli(){
    $data = array(
            'id' => $this->input->post('id'),
      'name' => $this->input->post('nama'),
      'price' => $this->input->post('harga'),
      'gambar' => $this->input->post('gambar'),
      'qty' =>$this->input->post('qty')
      );
  $this->cart->insert($data);
    redirect('Shopping_cart');
  }

  public function keranjang_belanja(){
    $data['cart'] = $this->cart->contents();
    $this->load->view('keranjang_belanja',$data);
  }

}


Sedikit penjelasan saja pada function index() pertama kita tampilkan data produk yang ada di database menggunakan function get() pada model M_cart.php.



$data['product'] = $this->M_cart->get()->result();


Dikarenakan kita akan menampilkan jumlah barang yang ada di dalam cart maka kita gunakan fungsi yang ada pada library cart untuk menampilkan barang apa saja yang ada di dalam cart yang kemudian akan kita hitung jumlah nya menggunakan fungsi count() yang akan kita buat pada view nanti.



$data['cart'] = $this->cart->contents();


Lalu pada function beli() kita gunakan untuk menyimpan barang yang ingin dibeli kedalam cart. Untuk memasukannya kita simpan terlebih dahulu data yang dikirim dari form pada view yang akan kita buat nanti kedalam array(). -tutorial membuat shopping cart menggunakan codeigniter



$data = array(
            'id' => $this->input->post('id'),

      'price' => $this->input->post('harga'),
      'gambar' => $this->input->post('gambar'),
      'qty' =>$this->input->post('qty')
      );


Kemudian kita masukan data yang ada didalam array tadi kedalam cart menggunakan fungsi insert() lalu kita alihkan halaman ke controller Shopping_cart menggunakan fungsi redirect().



$this->cart->insert($data);
    redirect('Shopping_cart');


Selanjutnya pada function keranjang_belanja() kita gunakan untuk menampilkan barang apa saja yang ada didalam cart menggunakan fungsi contents().



$data['cart'] = $this->cart->contents();


 

Membuat Model Baru

Setelah kita membuat sebuah controller baru sekarang kita buat sebuah model baru dengan nama M_cart.php. Kemudian kita buat sebuah function baru dengan nama get(). Function ini akan kita gunakan untuk menampilkan daftar produk yang ada didalam database kita. Silahkan salin baris kode berikut kedalam model yang sudah sobat buat tadi.




<?php
defined('BASEPATH') OR exit('No direct script access allowed');

class M_cart extends CI_Model{

  public function __construct()
  {
    parent::__construct();
  }

  public function get(){
    return $this->db->get('produk');
  }

}


 

Membuat 2 Buah View Baru

Kemudian setelah kita membuat sebuah controller dan model baru langkah terakhir kita buat 2 buah view baru dengan nama index.php yang akan kita gunakan untuk menampilkan data produk yang ada didalam database dan keranjang_belanja.php yang akan digunakan untuk menampilkan data produk yang kita simpan di dalam cart.

Sebelum kita membuat view baru silahkan sobat buat sebuah folder baru dengan nama assets dan didalamnya buat 2 buah folder baru dengan nama bootstrap dan images. Folder bootstrap digunakan untuk menyimpan resource bootstrap nya karena kita akan menggunakan bootstrap untuk mempercantik tampilannya, dan folder images digunakan untuk menyimpan gambar yang dibutuhkan oleh data produk yang ada di dalam database kita.
